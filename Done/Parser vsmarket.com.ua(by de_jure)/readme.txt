Установка:
1. Разархивировать содержимое в удобную для вас папку
2. На PC должен быть установлен браузер Mozilla Firefox(желательно последней версии)
3. Включите VPN. Парсер работает абсолютно без прокси, благодаря чему может спарсить всю информацию без
смены IP-адреса(это показали проведённые тесты, в случае каких либо проблем - напишите мне в Telegram).
3. Запуск скрипта производится через script.exe, остальные файлы - вспомогательные.
4. После первого запуска script.exe желательно его закрыть, и открыть снова.
5. Следовать инструкциям в консоли скрипта.
6. После того, как скрипт начнёт парсить страницы, его и браузер можно смело свернуть.


Если вам по каким-либо причинам нужно экстренно остановить работу парсера: сначала закрываете окно script.exe, и только
после этого браузер Firefox. В .json файле вы обнаружите столько товаров, сколько скрипт успел спарсить. В следующий раз он
начнёт всю работу с нуля.
